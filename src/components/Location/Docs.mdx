# Location

An interactive Google Map component that renders a single-location marker with an info window or a set of multiple markers with click-to-reveal labels.

## When to use

- Display a campus building, event venue, or office address on a map
- Show multiple related locations (e.g. campus buildings) with interactive markers
- Embed a map alongside contact or event details
- When the consumer has a Google Maps API key and coordinates to provide

## Import

```tsx
import { Location, type LocationProps, type MarkerData } from '@cuweb/rds2';
```

## Basic usage

```tsx
// Single marker
<Location
  lat="45.3875812"
  lng="-75.6982089"
  location="Carleton University Raven's Nest"
/>

// Multiple markers
<Location
  center={{ lat: '45.3850225', lng: '-75.6946679' }}
  markers={[
    { id: '1', name: 'Carleton University', position: { lat: 45.3850225, lng: -75.6946679 } },
    { id: '2', name: "Raven's Nest", position: { lat: 45.3875812, lng: -75.6982089 } },
  ]}
/>
```

## CSS classes

| Class | Condition |
|---|---|
| `cu-location` | Always applied to the root `<div>` |
| `cu-location--rounded` | Applied when `isRounded` is `true` (default) |
| `cu-location__map` | Applied to the `GoogleMap` container |
| `cu-location__info` | Applied to the info window content wrapper |
| `cu-location__info-link` | Applied to the "View on Google Maps" anchor |

## Design tokens

| Token | Controls |
|---|---|
| `--rds--radius-lg` | Border radius when `cu-location--rounded` is active |
| `--rds--font-size-base` | Font size of info window content |
| `--rds--color-primary` | Link color in info window |
| `--rds--color-primary-dark` | Link hover color in info window |

## Accessibility

- The root `<div>` receives `role="region"` and `aria-label` set to the `location` prop value (or `"Map"` as a fallback), making it a named landmark for screen readers
- Marker clicks open info windows that contain the location name and a text link; both are keyboard accessible through the Google Maps embedded UI
- The "View on Google Maps" link opens in a new tab with `rel="noreferrer"` to prevent referrer leakage
- Map controls (zoom, fullscreen, street view) are provided by Google Maps and are keyboard accessible by default
