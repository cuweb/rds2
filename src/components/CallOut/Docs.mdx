# Call Out

A bordered content block used to draw attention to a key message, CTA, or event. Renders a heading via `PageHeader` followed by arbitrary content — typically a paragraph and a `ButtonGroup`.

## When to use

- Highlighting an important action (apply, register, donate)
- Promoting an upcoming event inline within a page
- Breaking up long content pages with a focused CTA block

## Import

```tsx
import { CallOut } from '@cuweb/rds2';
```
