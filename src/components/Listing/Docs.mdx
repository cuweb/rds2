# Listing

An individual list item component designed for use inside `StackedList`. Each `Listing` stacks its visual element (figure, date thumb, or icon) above a body area containing the title, excerpt, and metadata — mirroring the `Card` layout but rendering as a `<li>` within the list container. Titles are linked directly; no separate call-to-action button is needed.

## When to use

- Inside a `StackedList` to display repeating content items such as news articles, events, people, or resource links.
- When content needs a consistent horizontal layout with a visual anchor (image, date, or icon) alongside a title, excerpt, and call-to-action.
- When items share a similar data shape but may vary in which visual thumb they use.
- As an alternative to `Card` when content is better presented in a compact list format. The linked title replaces a standalone call-to-action button.

## Import

```tsx
import { Listing } from '@cuweb/raven-design-system';
```

## Basic usage

```tsx
<StackedList cols="2">
    <Listing>
        <Listing.Figure>
            <img src="..." alt="..." width="400" height="300" />
        </Listing.Figure>
        <Listing.Body>
            <Listing.Header title="Article title" link="/article" date="2026-01-15 10:00:00" />
            <Listing.Excerpt text="Short description of the article." />
        </Listing.Body>
    </Listing>
</StackedList>
```

## Accessibility

- `Listing` renders a semantic `<li>` element; place it inside `StackedList` which renders a `<ul>` by default.
- `Listing.DateThumb` is `aria-hidden="true"` — the same date information is provided accessibly by `Listing.EventMeta` which uses proper `<time>` elements with `dateTime` attributes.
- `Listing.IconThumb` is `aria-hidden="true"` — it is a decorative visual and the title conveys the item's purpose.
- `Listing.Header` wraps the title in a `<h2>` by default; use the `as="h3"` prop when the listing appears inside an existing `<h2>` section heading to maintain heading order.
- `Listing.Header` uses `useLinkContext()` to render links, enabling consumers to swap in Next.js `Link` or React Router `Link` for client-side routing without changing the component API.
- `Listing.Initials` includes an empty `alt=""` on the background image so screen readers skip the decorative grey image.
- Pass meaningful `alt` text on images inside `Listing.Figure` to describe the visual content.
