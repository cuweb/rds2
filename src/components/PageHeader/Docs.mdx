# Page Header

The primary heading block for content pages. Renders a semantic `<header>` element with a heading, optional pre-header label, and an optional description.

## When to use

- Top of any content page to establish the page title
- Profile pages with a name and pronoun
- Section headers where a secondary h2 heading is needed

## Import

```tsx
import { PageHeader } from '@cuweb/raven-design-system';
```
