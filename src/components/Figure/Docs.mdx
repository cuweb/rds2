# Figure

An image or media wrapper with optional caption, float alignment, and border radius. Renders a semantic `<figure>` element around any image or media content.

## When to use

- Inline images within article or body copy that need float-left or float-right layout
- Full-width images that benefit from a shadow and rounded corners
- Any image that carries a descriptive caption

## Import

```tsx
import { Figure } from '@cuweb/raven-design-system';
```

## Props

| Prop       | Type                                   | Default | Description                                       |
| ---------- | -------------------------------------- | ------- | ------------------------------------------------- |
| `children` | `ReactNode`                            | —       | Image or media content                            |
| `caption`  | `string`                               | —       | Optional caption rendered in a `<figcaption>`     |
| `size`     | `xs \| sm \| md \| lg \| full`         | `full`  | Max-width constraint (activates at lg+ for xs–lg) |
| `align`    | `none \| left \| right \| center`      | `none`  | Float alignment; floats activate at lg+           |
| `rounded`  | `none \| sm \| md \| lg \| xl \| full` | `none`  | Border radius on the inner wrapper                |
