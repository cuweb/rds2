## Packages

`@cuweb/raven-design-system` is a private package on GitHub Packages. It includes components, tokens, styles, and Font Awesome Pro icon components.

## Quick start

```sh
# 1. One-time per machine: GitHub Packages auth
#    gh auth refresh --scopes read:packages
#    echo "//npm.pkg.github.com/:_authToken=$(gh auth token)" >> ~/.npmrc

# 2. In your project
echo "@cuweb:registry=https://npm.pkg.github.com" >> .npmrc
pnpm add @cuweb/raven-design-system
```

```tsx
import { Button, Icon } from '@cuweb/raven-design-system';
import '@cuweb/raven-design-system/styles';

<Button title="Save" icon="floppy-disk" />;
```

For the full install flow with troubleshooting, see [Installation](?path=/docs/overview-getting-started-installation--docs).
